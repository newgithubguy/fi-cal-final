## Previous Feature Updates (v1.2.0)

**Account Management:**
- Assign custom colors to accounts by clicking the 🎨 button - choose from 10 preset colors or no color.
- Color-coded account indicators in the sidebar make it easy to visually distinguish accounts at a glance.

**Payee & Description Management:**
- Click the 📝 button to open the Payee & Description manager.
- Organize and clean up your transaction entry history.
- Edit any payee or description entry by clicking the ✎ button.
- Delete unwanted payees or descriptions by clicking the × button.
- Switch between Payees and Descriptions tabs to manage each separately.

## Previous Updates (v1.1.1)

- Enter transaction amounts as positive numbers and choose Expense or Income from a dropdown.
- See the selected transaction type more clearly with red Expense and green Income labels.
- Edit transactions with the same Income / Expense workflow used when adding them.
- Work in a cleaner layout with Add Transaction above Transactions.
- Click a calendar day and, if it is not today, the selected day now turns white for better visibility.
- Read clearer field labels with Payee / Description in the transaction forms.
- Benefit from improved recurrence handling, including proper support for one-time and daily entries.

## Why It Matters

- Faster transaction entry with fewer sign mistakes and better account organization.
- Better visual feedback in both the form, calendar, and account list.
- More consistent behavior between the add and edit workflows.
- Cleaner server-side recurrence handling with fewer false warnings.
- Full control over your transaction history with edit and delete capabilities.
